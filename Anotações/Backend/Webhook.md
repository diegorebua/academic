# Webhook

## Conceito rápido: 
A analogia da encomenda ou do "Don't call us, we'll call you"

## O que é?
É uma notificação automática enviada de um sistema para o outro. Tecnicamente, é uma requisiçao HTTP POST contendo um JSON (payload) enviada para uma URL específica do seu servidor assim que um evento ocorre.

## Definição: 
Um Webhook é uma notificação automática enviada de um sistema para outro quando um evento ocorre. Tecnicamente, é uma requisição HTTP POST que um serviço externo (ex: Stripe, GitHub) faz para uma URL específica (endpoint) do seu servidor, enviando um corpo de dados (payload JSON).
